# Create NFTs on Forest

## Create collection

Creating NFT collections on the aelf blockchain requires you to possess a SEED NFT corresponding to the symbol you want. So make sure you have a SEED beforehand.

### 1. Obtain SEED

Refer to [Obtain SEED](docs/QuickStartGuides/LaunchTokenAndNFTCollection/ObtainSEED.mdx)

### 2. Create

Visit [Forest](https://test.eforest.finance/) first, click the wallet icon and then "Connect" your wallet account. You can log in to an existing account using wallet extensions, Web2 info, or scanning the code. If you don't have an account, you can sign up through the same methods.

<div style={{display: 'flex', justifyContent: 'space-around', alignItems: 'start', width: '100%'}}>
  <img src='/img/quick-start/forest-hero.png' width={'72%'} style={{border: '1px solid #dee2e8'}} />
  <img src='/img/quick-start/forest-login.png' width={'27%'} style={{border: '1px solid #dee2e8'}} />
</div>

After logging in, click "Create" and then "Create a Collection." Please note that you can't create NFT items without owning a collection in your account.

<div style={{display: 'flex', justifyContent: 'center', width: '100%'}}>
  <img src='/img/quick-start/forest-create.png' width={500} style={{border: '1px solid #dee2e8'}} />
</div>

Then upload the logo and featured image, and enter/select required fields like collection name, symbol, blockchain, burning status. You can also customise other fields to associate this NFT collection with more info.

<div style={{display: 'flex', justifyContent: 'center', width: '100%'}}>
  <img src='/img/quick-start/forest-create-collection-form.png' width={500} style={{border: '1px solid #dee2e8'}} />
</div>

After clicking "Create," you need to wait a second for the transaction to be confirmed on the chain.

<div style={{display: 'flex', justifyContent: 'center', width: '100%'}}>
  <img src='/img/quick-start/forest-create-collection-process.png' width={500} style={{border: '1px solid #dee2e8'}} />
</div>

When it's created, you can find the NFT collection in "My Collections."

<div style={{display: 'flex', justifyContent: 'center', width: '100%'}}>
  <img src='/img/quick-start/forest-my-collection.png' width={500} style={{border: '1px solid #dee2e8'}} />
</div>

# Create NFT

NFT can only be created within NFT collections, so make sure you have an NFT collection beforehand. Once the preparation has been done, click "Create an Item" in Forest.

<div style={{display: 'flex', justifyContent: 'center', width: '100%'}}>
  <img src='/img/quick-start/forest-create-item.png' width={500} style={{border: '1px solid #dee2e8'}} />
</div>

Then upload the NFT file, JPG, PNG, GIF, MP3, and MP4 supported, and then enter/select required fields like name, collection, token ID and quantity. You can also customise other fields to associate this NFT with more info.

<div style={{display: 'flex', justifyContent: 'center', width: '100%'}}>
  <img src='/img/quick-start/forest-create-item-step.png' width={500} style={{border: '1px solid #dee2e8'}} />
</div>

After clicking "Create," you need to wait a second for the transaction to be confirmed on the chain.

<div style={{display: 'flex', justifyContent: 'center', width: '100%'}}>
  <img src='/img/quick-start/forest-create-item-process.png' width={500} style={{border: '1px solid #dee2e8'}} />
</div>

When it's created, you can find the NFT item in your NFT collection.

If you want, you can create more items under this collection or you can also creat more collections using different SEEDs.
