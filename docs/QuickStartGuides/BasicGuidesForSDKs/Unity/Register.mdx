### Usage

```typescript
var param = new RegisterParams
{
    type = AccountType.Email, 
    loginGuardianIdentifier = "loginGuardianIdentifier",
    chainId = "chainId",
    verifierId = "verifierId",
    extraData = "extraData",
    verificationDoc = "verificationDoc",
    signature = "signature"
};
StartCoroutine(DID.Register(param, result =>
{
    // logged in
}, OnError));
```
